# DXF Samples

このディレクトリには、DXFの仕様を理解し、パーサーのテストに使用するためのサンプルファイルが格納されています。

| ファイル名 | バージョン | 解説 |
| :--- | :--- | :--- |
| `minimal_r12.dxf` | R12 (AC1009) | 最小構成のDXF。ENTITIESセクションに1本のLINEのみ。 |
| `simple_shapes.dxf` | 2000 (AC1015) | LINE, CIRCLE, ARC, TEXTなど、基本的なエンティティのセット。 |
| `lwpolyline_example.dxf` | 2000 (AC1015) | バルジ（円弧）を含むLWPOLYLINEの例。 |
| `block_example.dxf` | 2000 (AC1015) | ブロック定義(BLOCK)とその参照(INSERT)を含む例。 |

## 使い方

1. **テキストエディタで開く**: グループコードと値のペアがどのように並んでいるかを確認してください。
2. **CADビューワーで開く**: 実際の描画結果を確認してください。
3. **自作パーサーの入力にする**: 期待した座標やプロパティが抽出できるかテストしてください。

## 注意事項
これらのファイルは学習・テスト用であり、複雑な実務図面のすべてのケースを網羅しているわけではありません。エッジケースのテストには、さらに大規模なファイルや、バイナリ形式のファイルが必要になる場合があります。
